# 📝 Online Complaint System

## 📌 Project Overview
The Online Complaint System is a web-based application developed using 
Java, Spring Boot, and MySQL. It allows users to register complaints 
and enables administrators to manage and track complaint status.

## 🚀 Tech Stack
- Java
- Spring Boot
- MySQL
- Maven
- REST APIs
- Git & GitHub

## ✨ Features
- User complaint registration
- View complaint status
- Admin complaint management
- Update complaint status
- Database integration using JPA
- RESTful API architecture

## 🛠 Installation & Setup

### 1️⃣ Clone Repository
git clone https://github.com/NandigariRupika/online-complaint-system.git

### 2️⃣ Open in VS Code / IntelliJ

### 3️⃣ Configure MySQL Database
Update application.properties with your database credentials.

### 4️⃣ Run Application
mvn spring-boot:run

Application runs at:
http://localhost:8080
